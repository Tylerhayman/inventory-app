END GOAL: 
An inventory app that allows the user to view the full inventory, a speicifc item in the inventory and with a form that allows the user to add or delete an item from the inventory.

The backend should have an Item sequelize that has:
- Name for each item
- Description for each item
- Category for each item
- Image for each item

You should be able to FETCH, GET, PUSH with the sequelize and the form together to make an inventory system.
Make an API holding the items (sequelize)

The frontend should have:
- An item list with all items from the sequelize
- The ability to click on an indivudual item from the main list
- A form to add items
- A form to edit items
- A form to delete items

The project should adhere and work with all tests.
